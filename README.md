# HR Analytics Dashboard | Tableau

An interactive Human Resources Analytics Dashboard built using Tableau Desktop to support workforce analysis and data-driven HR decision-making. The project transforms employee data into actionable insights through workforce overview metrics, demographic analysis, salary comparisons, and detailed employee records.

## Project Overview

This project was completed as part of the Data and Artificial Intelligence track in the Cyber Shujaa Program. The objective was to simulate a real-world HR analytics project by analyzing employee data and developing an interactive dashboard that supports organizational decision-making.

The dashboard provides insights into workforce composition, hiring and termination trends, employee demographics, compensation patterns, and detailed employee records.

## Business Problem

Human Resource departments require accurate workforce insights to support strategic planning, talent management, diversity initiatives, and compensation analysis.

This dashboard was designed to help HR managers answer key questions such as:

- How many employees are currently active?
- What are the hiring and termination trends over time?
- What does the workforce demographic profile look like?
- How does salary vary across departments, education levels, and gender?
- Where are employees geographically distributed?
- Which workforce segments require attention?

## Objectives

- Understand HR business requirements and reporting needs.
- Prepare and transform employee data for analysis.
- Create calculated measures to support workforce analytics.
- Design interactive visualizations using Tableau.
- Build a comprehensive HR dashboard.
- Enable dynamic exploration through filters and drill-down capabilities.

## Project Workflow

### Business Understanding

The project began by analyzing HR reporting requirements and identifying key workforce metrics required by management.

The dashboard was designed around four primary analytical areas:

- Workforce Overview
- Employee Demographics
- Income Analysis
- Detailed Employee Records

### Data Preparation and Transformation

The HR dataset was imported into Tableau and prepared for analysis through:

- Data quality inspection
- Field type standardization
- Date formatting
- Category harmonization
- Geographic hierarchy creation
- Data cleaning and validation

These steps ensured consistency and accuracy across all dashboard views.

### Calculated Measures

Several calculated fields were created to support workforce analysis, including:

- Employee Age
- Active Employee Count
- Terminated Employee Count
- Average Salary
- Workforce Demographic Metrics
- Department-Level Performance Indicators

These calculations enabled deeper insights into employee trends and organizational structure.

### Visualization Design

A variety of visualization techniques were used to communicate insights effectively:

#### Workforce Overview

- KPI Cards
- Hiring Trend Analysis
- Termination Trend Analysis
- Department Distribution Charts
- Job Title Analysis

#### Demographics Analysis

- Gender Distribution
- Age Group Analysis
- Education Level Breakdown
- Geographic Employee Distribution Maps

#### Income Analysis

- Salary Comparison by Gender
- Salary Comparison by Education Level
- Salary Analysis by Department
- Age vs Salary Scatter Plots

All visualizations were designed with interactivity to enable dynamic exploration of workforce data.

### Dashboard Development

The final dashboard was organized into three key sections:

#### Overview

Provides a high-level summary of workforce metrics and organizational trends.

#### Demographics

Analyzes workforce composition through gender, age, education, and location perspectives.

#### Income Analysis

Explores compensation patterns and salary distribution across workforce segments.

Interactive filters allow users to explore data by:

- Department
- Gender
- Location
- Education Level
- Employment Status

### Employee Records View

A detailed employee table was created to support individual-level analysis.

The table includes:

- Employee Name
- Department
- Position
- Gender
- Age
- Education
- Salary
- Employment Information

Users can filter records dynamically to investigate specific employee groups or organizational segments.
